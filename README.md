# Descargador de Videos — Windows

Descargador de videos con interfaz gráfica profesional basada en CustomTkinter. 
Desarrollado para funcionar en sistemas Windows de 32 y 64 bits.

## 🛠️ Guía para Desarrolladores

### Requisitos previos
Para preparar tu entorno de desarrollo, asegúrate de tener Python 3.12 instalado y ejecuta:

pip install -r requirements.txt

### Ejecución y Empaquetado
- Modo desarrollo: python main.py
- Generar ejecutable: python empaquetar.py
  (El ejecutable se genera en la carpeta dist/DescargadorVideos/)

## 🚀 Distribución y Releases
1. Ve a la pestaña Releases en este repositorio.
2. Crea una New Release.
3. Sube el archivo .zip que contiene tu ejecutable generado.

## 📥 Guía de Instalación para Usuarios
1. Descarga el archivo .zip desde la sección de Releases.
2. Descomprime el archivo en cualquier carpeta de tu computadora.
3. Haz doble clic en DescargadorVideos.exe para iniciar la aplicación.

## ⚙️ Configuración
- Contenido privado (Instagram/Facebook): Coloca un archivo cookies.txt en la misma carpeta donde está el ejecutable.
